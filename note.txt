- git init: khởi tạo git cho 1 project (chỉ dùng một lần duy nhất)
- git add: Săp xếp file đã bị thay đổi, thêm mới/xóa so với lần commit trước đó, để chuẩn bị cho lần commit tiếp theo.
- git add filename1 filename2: thêm một hoặc nhiều file bị thay đổi
- git add .: thêm toàn bộ file bị thay đổi
- git commit: xác nhận & lưu trữ các file thay đổi
    + git commit -m "message conmmit"


    - Phím tắt: control + "`"

- git log: xem lại lịch sử commit
- git status: xem trạng thái hiện tại của thư mục làm việc, hiển thị những thay đổi so với lần commit trước đó, đồng thời hiển thị trạng thái của các fike đó như là đã được xác nhận đã commit hay chưa

- git config --global user.name "user name"
- git config --global user.email "Email name"


-git push: đẩy tất cả commit lên trên remote repository
